# **App Name**: connLevel Hub

## Core Features:

- Dashboard Overview: Provides a visual representation of key metrics across various categories, such as PrP+PoP activations, B2C contracts, and MNP performance.
- Outlet Performance Analysis: Enables detailed examination of individual outlet performance against defined levels, highlighting achievements and areas for improvement.
- Data Import and Processing: Allows users to upload .xlsx or .xls files, which are then parsed and processed to extract relevant data for report generation.
- Proactive alert generator: LLM tool suggests personalized actions based on store type and its recent sales trends. Tool will look at the extracted outlet-specific achievements and potential opportunities for improvement.
- Customizable Filtering: Offers filtering options on the B2C Contracts (PoP) report. Filter the outlets, connection types and logins to see detailed information.
- Achievement tracking: Show each outlet's current sales achievements

## Style Guidelines:

- Primary color: Deep, intense blue (#2E3192) evoking focus and depth.
- Background color: Near-black (#121214), enhancing contrast and readability in a dark theme.
- Accent color: Soft violet (#B150B1), provides highlights on actionable items.
- Body and headline font: 'Inter', a sans-serif font known for its modern and neutral design, is used for both headlines and body text to maintain consistency and readability.
- Use minimalist icons, primarily line icons, in a light color that stands out against the dark background. Icons should be related to data analysis and performance metrics.
- Use a grid-based layout to present data in a structured and easily digestible format. The layout should be responsive to accommodate different screen sizes.
- Incorporate subtle animations and transitions to provide feedback on user interactions and guide the user through the interface. Avoid excessive animations that could distract the user.
